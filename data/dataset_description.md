# 网球比赛动量分析数据集描述

## 数据集基本信息

### 数据范围
- **时间范围**: 2011年1月1日 - 2017年12月31日
- **比赛类型**: ATP（男子职业网球）巡回赛
- **比赛级别**: 仅限Main Draw（正赛）
- **赛制类型**: 三盘两胜制（不含大满贯赛事）
- **地理范围**: 全球ATP巡回赛赛事

## 原始数据文件

### 文件列表
| 文件名 | 大小 | 行数 | 描述 |
|--------|------|------|------|
| pbp_matches_atp_main_archive.csv | ~3,109 KB | 10,902 | 2011-2015年历史数据 |
| pbp_matches_atp_main_current.csv | ~627 KB | 2,148 | 2015-2017年最新数据 |

### 文件结构
```
原始数据文件 (Raw Data Files)
├── 归档数据 (2011-2015)
│   ├── 正赛数据 (pbp_matches_atp_main_archive.csv) ✓
│   └── 资格赛数据 (pbp_matches_atp_qual_archive.csv)
└── 当前数据 (2015-2017)
    ├── 正赛数据 (pbp_matches_atp_main_current.csv) ✓
    └── 资格赛数据 (pbp_matches_atp_qual_current.csv)
```

> **注**：本项目仅使用标记✓的正赛数据文件

## 数据字段详细说明

### 1. 基础信息字段

| 字段名 | 数据类型 | 是否必需 | 描述 | 示例值 |
|--------|----------|----------|------|--------|
| `date` | String | ✓ | 比赛日期（YYYY-MM-DD格式） | "2013-01-01" |
| `tournament` | String | ✓ | 赛事名称 | "Brisbane International" |
| `tour` | String | ✓ | 巡回赛类型 | "ATP" |
| `draw` | String | ✓ | 比赛阶段 | "Main" |
| `server1` | String | ✓ | 第一盘先发球的选手 | "Roger Federer" |
| `server2` | String | ✓ | 第一盘后发球的选手 | "Rafael Nadal" |
| `winner` | Integer | ✓ | 比赛获胜者（1=server1, 2=server2） | 1 |
| `score` | String | ✓ | 最终比分 | "6-4 6-7(5) 6-3" |
| `adf_flag` | Integer | ✓ | Ace/双误标记 | 1 |

### 2. 核心数据字段（逐分数据）

| 字段名 | 数据类型 | 长度 | 描述 | 示例 |
|--------|----------|------|------|------|
| `pbp` | String | 可变 | 逐分数据编码字符串 | "S;R;A;D;S;R.;S;R" |

#### pbp字段编码规则
| 字符 | 全称 | 含义 | 计分规则 |
|------|------|------|----------|
| **S** | Server won | 发球方得分 | 发球方+1分 |
| **R** | Returner won | 接发方得分 | 接发方+1分 |
| **A** | Ace | 发球直接得分 | 发球方+1分（特殊标记） |
| **D** | Double fault | 双发失误 | 接发方+1分（特殊标记） |
| **;** | Game delimiter | 局结束 | 重置局内比分 |
| **.** | Set delimiter | 盘结束 | 重置盘内比分 |
| **/** | Service change | 抢七发球交换 | 仅抢七局有效 |

### 3. 比分格式规范
```
标准格式: "[盘1比分] [盘2比分] [盘3比分]"
示例1: "6-4 6-3"            # 两盘比赛
示例2: "6-4 4-6 6-2"        # 三盘比赛
示例3: "7-6(5) 6-4"         # 包含抢七
```
